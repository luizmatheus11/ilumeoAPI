## Instalação

```bash
$ npm install

# Criar o docker
$ docker compose up

# Setup Prisma
$ npx prisma generate

#Configurar o banco de dados
$ npx prisma db push

# Popular o banco de dados
$ npx prisma db seed
```
## Rodar a aplicação

```bash
# build
$ npm run build

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
